# ABAP-Themes
My personal SAP GUI ABAP themes. **Don't forget to backup your old `abap_spec.xml` file.** I also use Consolas (size 10).

Theme location
- Windows: `C:\Users\USERNAME\AppData\Roaming\sap\sap gui\ABAP Editor\abap_spec.xml`
